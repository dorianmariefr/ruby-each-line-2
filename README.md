# ruby-each-line-2

## Description

Execute ruby on each line from input

## Installation

```
$ gem install ruby-each-line-2
```

## Usage

```
USAGE: INPUT | ruby-each-line CODE"
```

```
EXAMPLE: ls | ruby-each-line "puts line.split('.').last"
```
